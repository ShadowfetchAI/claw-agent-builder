# Winston Scientific Support

If you need help with Winston Scientific, use the contact details below.

## Contact
- Support email: RobertCorbin84@gmail.com

## What to include in your message
- your device model
- your iOS version
- a short description of the problem
- screenshots if the issue is visual

## Product scope
Winston Scientific is designed to work offline. If something appears broken, include the exact screen and math mode you were using:
- Basic
- Scientific
- Graphing
- Statistics
- Calculus
- Linear Algebra
- Math Books

## Response expectations
Support is handled directly by the developer.
